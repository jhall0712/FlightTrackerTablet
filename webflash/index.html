Flight Tracker Tablet

Flash the Waveshare ESP32-S3 7 inch tablet with the visual overhead flight tracker firmware.

Install Firmware

Use Chrome or Edge on desktop. Connect the tablet USB-C port that exposes the CH343 serial adapter.

After Flashing

  1. Join Wi-Fi network FlightTrackerTablet.
  2. Use password flighttablet.
  3. Open http://192.168.4.1.
  4. Save Wi-Fi, latitude, longitude, and radius.
